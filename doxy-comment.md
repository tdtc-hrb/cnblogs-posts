[Type one of the following symbols](https://www.jetbrains.com/help/rider/Creating_and_Viewing_Doxygen_Documentation.html#create-doxygen-comments): 
- [c-sharp style](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/xmldoc/recommended-tags)
```
///
```
- Qt style
```
//!
/*!
```
- javadoc
```
/**
```

#### Putting documentation after members
```
//!< this is End of line
```

### javadoc style
- [file](https://www.doxygen.nl/manual/commands.html#cmdfile)
```
/**
 * @file PokerType.h
 * Created by tdtc-hrb on 2/14/2026 - UTC+8.
 */
```
- [author](https://www.doxygen.nl/manual/commands.html#cmdauthor)
```
/**
 *  Pretty nice class.
 *  @details   This class is used to demonstrate a number of section commands; 
 *             detailed description.
 *  @author    John Doe
 *  @author    Jan Doe
 *  @version   1.16.1
 *  @date      2026-03-11
 */
```
